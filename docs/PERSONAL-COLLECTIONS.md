# Personal collections and reading lists

Scholarium separates a private reading library from discovery ranking.

## What is private

- Collection names, descriptions, membership, and saved-publication timestamps belong only to the signed-in account.
- A default **Reading list** is created only when a person first saves a public publication.
- A person can create additional named collections. Collection data is included in their private account export.

## What is not used for ranking

Saving a publication does not change its public reach, the author’s standing, topic visibility, search position, subscription state, or any other member’s feed. The discovery feed uses its documented explicit favourite/reaction inputs separately; a collection item is not a favourite signal.

## Availability and removal

Only currently public, eligible publications can be saved. If a saved publication later becomes private, quarantined, or removed, it no longer appears in the reading-list view. The private item record remains exportable so the owner can understand why a previous save is unavailable.

No collection is public, shareable, or monetized in this release.
