# Scholarium — publication formats

The Scholarium publication record accepts research articles, reports, syntheses, white papers, full books, presentations, datasets, school projects, software projects, Git trees, teaching artifacts and project updates. Each record receives the same account-bound provenance receipt and can carry validated document artifacts.

Video, Short and Live/replay records are currently **metadata-first**: a person can publish the explanatory record, transcript context and sources, but the Stream media handoff is intentionally marked pending. This prevents the interface from implying that video hosting or Live broadcasting is already active.

QuaNthoR remains a guide, not a gate. Its supported guide types are mapped to the closest publication record without rewriting the author’s content or blocking publication.
