# AGENTS.md

This is an official SecuredMe school-tool repository. Preserve the education-first boundary.

- Official AI-assisted school route: Codex/OpenAI or Antigravity/Gemini only.
- Do not add Ollama Cloud, uncensored local AI, or unknown agent providers as official classroom routes.
- Do not rewrite the maintained mechanism directly for a private user request.
- Build separate local forks, adapters, or plugins when a user wants a different mechanism.
- Keep publication, citation, and repository claims source-linked and non-authoritative unless validated.

Private modified copies are not a maintainer support obligation unless reviewed through PR.
